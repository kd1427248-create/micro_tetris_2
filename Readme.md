#ミクロ・テトリス (IoT Game)

MicroPythonで動く、縦持ち専用の本格テトリスです！
接地したあとにブロックを動かすと、固定されるまでの時間が少し伸びる「固着猶予システム」も搭載しています。

##🕹️ 操作方法（縦持ち時）
本体を縦に持ったときのボタンの役割です。

* **SW3 (Pin 6):** ブロックを **【左】** に動かす
* **SW1 (Pin 2):** ブロックを **【右】** に動かす
* **SW2 (Pin 3):** ブロックを **【回転】** させる
* **SW3 ＋ SW1 (同時押し):** **【ハードドロップ】**（一瞬で一番下まで落として固定する）

---

##🔌 必要な配線（ピン番号）
マイコンと各パーツを以下のように接続してください。

* **左移動ボタン (SW3):** Pin 6
* **右移動ボタン (SW1):** Pin 2
* **回転ボタン (SW2):** Pin 3
* **スピーカー:** Pin 21
* **フルカラーLED (NeoPixel):** Pin 10
* **OLEDディスプレイ:** SCL = Pin 9 / SDA = Pin 8

---

## 📦 必要なライブラリ
このゲームを動かすには、マイコン側に以下のライブラリファイルを入れておく必要があります。
1. `ssd1306.py` （OLEDディスプレイ用）
2. `neopixel.py` （フルカラーLED用 ※標準で入っている場合もあります）
